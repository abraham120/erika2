Scilab/Scicos support for ERIKA Enterprise and FLEX
---------------------------------------------------

This directory contains the needed software which has to be built
together with ERIKA Enterprise to build a working application which is
automatically generated by Scilab and Scicos.

The authors of this contribution is mainly the METALAU Project from 
INRIA (FR) http://www.scicos.org, with in particular additional 
contribution and customization from
- Simone Mannori - INRIA (FR)
- Roberto Bucher - SUPSI Lugano (CH)
- the Evidence Team

Please refer to the online documentation on the Evidence web site to
know how to use Scilab and Scicos to generate a working dsPIC
application.
